## 1. 프로젝트 소개
```
iOS Lifecycle 이해를 위해 여러 ViewController 간에 이동할 수 있는 간단한 앱 프로젝트
'해마'라는 일기 작성 어플리케이션을 참고하여 제작
```

## 2. 사용한 기술
- Modal을 통한 화면 전환
- 여러 View Controller설정
- Awesome iOS FSCalendar 사용

## 3. 설치 방법
1. 다운로드 후 압축을 푼다
2. .xcworkspace 확장자 파일 실행
